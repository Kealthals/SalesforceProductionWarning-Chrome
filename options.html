HOW TO USE:
1. By Default, all production's border is Red.
2. Of course default color could be changed.
3. You could specify different production org's color by filling in Custom Domain in the textbox.
   For Example.:
   In Classic, My org's url is  https://kealsalas.my.salesforce.com/
   Fill 'kealsalas' in the textbox, then select a color.

   In Lightning Experience, My org's url is  https://kealsalas.lightning.force.com/
   Fill 'kealsalas' in the textbox, then select a color.
   If the sandbox mode is enabled, the url is https://kealsalas--dev1.sandbox.lightning.force.com/
   Fill 'kealsalas--dev1' in the textbox, then select a color.
4. Switch On Change Tab's Icon Color Option to set the same color on tab's cloud icon. The tab need to be reloaded to take effect.
5. Switch On Overlay UI With Warning Stripes Option to experience the new feature.The tab need to be reloaded to take effect.
